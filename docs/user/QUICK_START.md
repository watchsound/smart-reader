# SmartReader Quick Start Guide

Get started with SmartReader in 5 minutes.

## Installation

1. Download SmartReader for your platform:
   - **Windows**: `SmartReader-Setup.exe`
   - **macOS**: `SmartReader.dmg`
   - **Linux**: `SmartReader.AppImage`

2. Run the installer and follow the prompts

3. Launch SmartReader

## Essential Workflow 1: Read and Annotate (2 minutes)

### Import Your First Book

1. Click **Bookshelf** (📚) in the sidebar
2. Click the **+** button
3. Select an EPUB or PDF file
4. Your book appears in the library

### Read and Highlight

1. Click the book cover to open
2. **Navigate**: Use arrow keys or click page edges
3. **Highlight**: Select text → choose color (yellow/green/blue/pink)
4. **Add Note**: Select text → click "Note" icon → type your note

**View your notes**: Right sidebar → "My Notes" tab

## Essential Workflow 2: Create Learning Plan (2 minutes)

### Start the Wizard

1. Click **Learning** (🎓) in sidebar
2. Click **"Create New Plan"**

### Follow the 5 Steps

**Step 1 - Goal**:
- Name: "My Vocabulary"
- Domain: Vocabulary
- Click Next →

**Step 2 - Material**:
- Choose "Manual Entry"
- Add test words: `ephemeral → lasting a short time`
- Or upload CSV: `word,definition`
- Click Next →

**Step 3 - Review**:
- Check items look correct
- Click Next →

**Step 4 - Commitment**:
- Daily time: 20 minutes
- Preferred time: Evening
- Click Next →

**Step 5 - Create**:
- Review settings
- Click "Create Plan"

## Essential Workflow 3: Study Session (1 minute)

### Start Studying

1. Click **"Study"** on your plan
2. **Front of card** shows question
3. Press **SPACE** to flip
4. **Rate your answer**:
   - **1** = Don't know (Again)
   - **2** = Hard (barely remembered)
   - **3** = Good (got it right)
   - **4** = Easy (knew instantly)

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Flip card |
| `1-4` | Rate answer |
| `H` | Get hint |
| `Esc` | End session |

### Complete Session

- Continue until all cards reviewed
- See your summary: accuracy, time, streak

## Configure AI (Optional but Recommended)

SmartReader works great without AI, but AI features unlock powerful capabilities.

### Quick Setup

1. Go to **Settings** (⚙️) → **AI Providers**
2. Choose a provider:
   - **Free**: Ollama (local, requires installation)
   - **Paid**: OpenAI, Claude, Gemini
3. Enter API key (for paid providers)
4. Click **"Test Connection"**
5. Choose default model

### What AI Enables

- **Smart explanations**: Select text → "AI Explain"
- **Smart summaries**: Auto-generated summaries
- **Vocabulary help**: Automatic definitions
- **Quiz generation**: Create practice questions
- **Translation**: 5-step translation learning

**Detailed setup**: See [AI Provider Setup Guide](AI_PROVIDER_SETUP.md)
